# 项目记忆 - html_XiaoYouXi

## 项目概述
给孩子做的 HTML 小游戏集合，卡通水彩风格。

## 项目结构
- `image/Yu_1.png` — 卡通水彩小鱼参考图
- `aquarium_octopuses_2026_7_05.html` — 早期水族馆章鱼游戏（已重构）
- `octopus-game/` — 章鱼游戏（海底大冒险）
  - `index.html` — 单文件 HTML5 Canvas 游戏
  - `assets/` — 图像资源（AI 生成的章鱼/水母/鲨鱼/海底背景 + 复用的鱼图）
  - 玩法：玩家控制鲨鱼（WASD），无尽模式，鲨鱼吃章鱼/水母/鱼群得分
  - AI：章鱼水母躲避鲨鱼并追鱼；鱼群平稳游动不逃跑；实体可游出屏幕从边缘重生
  - 数量：3章鱼+2水母+30鱼+1玩家鲨鱼
  - 速度：鲨鱼(5.0) > 章鱼/水母(3.0) > 鱼群(1.5)
  - 积分：仅鲨鱼吃猎物得分（章鱼/水母吃鱼不加分）

## 约定
- 单 HTML 文件，内嵌 CSS/JS，孩子双击即可打开
- 图像风格：卡通水彩，透明背景 PNG，与 Yu_1.png 一致
- 中国股市配色惯例（红涨绿跌）适用于金融相关工具
